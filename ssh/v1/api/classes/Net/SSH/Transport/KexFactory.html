Net::SSH::Transport::KexFactory
In: lib/net/ssh/transport/kex.rb
Parent: Object

A singleton factory for providing services (objects) that implement various key-exchange (kex) algorithms. Currently, only "diffie-hellman-group1-sha1" is supported.
