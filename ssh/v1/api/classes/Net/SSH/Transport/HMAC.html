Net::SSH::Transport::HMAC

A wrapper class for an HMAC implementation. This just delegates to the OpenSSL::HMAC#digest method.

Public Class methods
new( digest, key, mac_length )

Create a new HMAC wrapper that will use the given digest, key, and number of bytes for the digest. If digest is nil, then there is no underlying HMAC algorithm, and digest will always return the empty string.
