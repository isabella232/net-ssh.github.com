Class
Net::SSH::Service::Channel
In: lib/net/ssh/service/channel.rb
Parent: Object

A channel is a single pseudo-connection, multiplexed onto the containing session connection. It allows data to be passed back and forth for a specific purpose (such as execution of remote commands), effectively in parallel with other channels, all on the same connection.

new( connection, channel_type, extra_data=nil, &on_confirm )

Create a new channel on the given connection, with the given channel-type. This will return immediately; the caller should call Connection#process_connection on the given connection in order to cause the channel to be processed. When the channel has been confirmed by the server, the attached block will be executed.

request_pty( opts = {} )

Request that a pty be opened for this channel. Valid options are :term, :chars_wide, :chars_high, :pixels_wide, :pixels_high, :modes, and :want_reply. :modes is a Hash, where the keys are constants from Net::SSH::Service::Term, and values are integers describing the corresponding key.
