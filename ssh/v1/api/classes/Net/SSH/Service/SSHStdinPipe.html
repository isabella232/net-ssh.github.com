Class
Net::SSH::Service::SSHStdinPipe
In: lib/net/ssh/service/process.rb
Parent: Object

A specialized class for use by the Net::SSH::Service::Process.popen3 method. An instance of this class represents a means of writing data to an SSH channel. This class should never be instantiated directly; use the popen3 method instead.
