Class
Net::SSH::SFTP::FileAttributes
In: lib/net/ssh/sftp/attrs.rb
Parent: Object

A class representing the attributes of a single file. This is used by the SFTP subsystem both when retrieving files, and when creating files. This encapsulates attributes as returned by all versions of the SFTP protocol, and not all attributes are available under all protocols. If any attribute is not set, the corresponding property will be nil.
